# Ys Release Promo

**Source:** *POPCOM* June 1987

## Script

「イース」は、「ザナドゥ」や「ロマンシア」とはタイプが違うRPG。町や村、野原などで多くの人々と話して、冒険の目的を見付け、たくさんの冒険の目的をひとつひとつ達成していくことで、本当の冒険の目的、大きな謎を知っていくのです。  
　町、村、野原の他にも、山や森、不思議な謎に包まれた神殿、一寸先は闇のダンジョンなど、広い世界で冒険は様々な形で繰り広げられ、キミたちの果敢な冒険心を満たすはずです。  
　画面は、「ロマンシア」と同じように完全重ね合わせ処理をし、ディスプレイいっぱいにフルカラースクロール。文句のつけようのない美しいグラフィックに仕上がりました。  
　そんなグラフィックとともにゲームのムードを盛上げてくれるのが、胸踊るようなBGMの数々。時には軽快に、時には重厚に、時にはソフトに、ときには激しく‥‥、まるで自分の心を反映するかのようです。  
「イース」は、ゲーマーを選びません。RPGが得意なゲーマーでも、苦手なゲーマーでも、冒険してみたいゲーマーなら誰でも、すぐに楽しめます。  
　もっともっと、たくさんのゲーマーにワクワクするような冒険を楽しんでもらいたいのです。  
　苦手だ、とさけて通ったり、逃げ腰にならないで、まず、町にいる人々と話をしてみれば、最初の冒険はカンタンです。  
　でも、「イース」の冒険はひとつではありません。とてもたくさんの冒険があります。その数の多さに驚くかもしれません。  
　本当の目的‥‥それは、もちろん誰にも明かせません。  
　自分でたくさんの冒険を経験しながら、少しずつ謎を解きほぐし、自分自身の力で見つけてください。安易な考えではとうてい解けませんが、少しも難しくはありません。  
　自分自身の思っていることを正直に、普段の生活の中で生じる問題を解決していく時のように冒険すれば、自然に謎も解けていくことでしょう。  
　ひとつの冒険が終わるごとに、誰もがきっと、心が洗われ、澄んでくるようなさわやかな気分を味わうことでしょう。  
　ドロドロとした闘いだけのRPGは、もう卒業して、そろそろ奥の深い本物のRPGに挑戦してみてはどうかな。
